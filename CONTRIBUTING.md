# Contributing to counter-strike-tool

Thanks for your interest in counter-strike-tool! This document describes how to contribute to the project.

Main repository: [galeseniorwedge/counter-strike-tool-v2](https://github.com/galeseniorwedge/counter-strike-tool-v2)

## How to contribute

1. Fork the [repository](https://github.com/galeseniorwedge/counter-strike-tool-v2).
2. Create a feature branch: `git checkout -b feat/your-change`.
3. Commit your changes with a descriptive message.
4. Push and open a pull request.

## Reporting issues

If you found a bug, please open an issue at [https://github.com/galeseniorwedge/counter-strike-tool-v2/issues](https://github.com/galeseniorwedge/counter-strike-tool-v2/issues).
