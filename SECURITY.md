# Security Policy

## Supported versions

Only the latest release of [counter-strike-tool](https://github.com/galeseniorwedge/counter-strike-tool-v2/releases/latest) is actively maintained.

## Reporting a vulnerability

If you discover a security issue in counter-strike-tool, please open a private security advisory at https://github.com/galeseniorwedge/counter-strike-tool-v2/security/advisories/new instead of a public issue.

We try to respond within 72 hours.
